Detailed Description

A TypeNameMatchRequestor collects matches from a searchAllTypeNames query to a SearchEngine. Clients must subclass this abstract class and pass an instance to the char[] packageName, int packageMatchRule, char[] typeName, int typeMatchRule, int searchFor, IJavaSearchScope scope, TypeNameMatchRequestor nameMatchRequestor, int waitingPolicy, org.eclipse.core.runtime.IProgressMonitor monitor) method. Only top-level and member types are reported. Local types are not reported.

While TypeNameRequestor only reports type names information (e.g. package, enclosing types, simple name, modifiers, etc.), this class reports TypeNameMatch objects instead, which store this information and can return an org.eclipse.jdt.core.IType handle.

This class may be subclassed by clients.
