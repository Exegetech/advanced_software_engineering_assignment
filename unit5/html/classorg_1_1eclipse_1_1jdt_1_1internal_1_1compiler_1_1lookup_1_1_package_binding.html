◆ getVisibleFor()

PackageBinding org.eclipse.jdt.internal.compiler.lookup.PackageBinding.getVisibleFor ( ModuleBinding  module)

If this package is uniquely visible to 'module' return a plain PackageBinding. In case of a conflict between a local package and foreign package the plain local package is returned, because this conflict will more appropriately be reported against the package declaration, not its references. In case of multiple accessible foreign packages a SplitPackageBinding is returned to indicate a conflict.
