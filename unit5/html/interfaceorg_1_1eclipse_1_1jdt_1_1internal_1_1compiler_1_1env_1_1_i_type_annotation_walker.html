Detailed Description

A TypeAnnotationWalker is initialized with all type annotations found at a given element. It can be used to walk into the types at the given element and finally answer the actual annotations at any node of the walk.

The walker is implemented as immutable objects. During the walk either new instances are created, or the current instance is shared if no difference is encountered.
