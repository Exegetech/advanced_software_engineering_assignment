Detailed Description

A SourceIndexer indexes java files using a java parser. The following items are indexed: Declarations of:

  • Classes
  • Interfaces;
  • Methods;
  • Fields;
  • Lambda expressions;
    References to:
  • Methods (with number of arguments);
  • Fields;
  • Types;
  • Constructors.
