Detailed Description

Modifier node. 

Modifier:
   public
   protected
   private
   static
   abstract
   final
   native
   synchronized
   transient
   volatile
   strictfp
   default

<p<blockquote>‍

The numeric values of these flags match the ones for class files as described in the Java Virtual Machine Specification (except for DEFAULT). Note that the Java model class org.eclipse.jdt.core.Flags also provides the same constants as this class.

Since
2.0 @noinstantiate This class is not intended to be instantiated by clients.
