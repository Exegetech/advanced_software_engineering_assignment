static IModule org.eclipse.jdt.internal.compiler.batch.ModuleFinder.extractAddonExport ( String  option)
staticprotected

Parses the –add-exports command line option and returns the package export definitions in the form of an IModule. Note the IModule returned only holds this specific exports-to clause and can't by itself be used as a module description.

The expected format is: –add-exports <source-module>/<package>=<target-module>(,<target-module>)*

Parameters
option
Returns
a dummy module object with package exports

static String[] org.eclipse.jdt.internal.compiler.batch.ModuleFinder.extractAddonRead ( String  option)
staticprotected

Extracts the single reads clause from the given command line option (–add-reads). The result is a String[] with two element, first being the source module and second being the target module. The expected format is: –add-reads <source-module>=<target-module>

Parameters
option
Returns
a String[] with source and target module of the "reads" clause.
