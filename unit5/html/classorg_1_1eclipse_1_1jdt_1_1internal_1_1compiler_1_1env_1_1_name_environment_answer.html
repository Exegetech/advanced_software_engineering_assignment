◆ getSourceTypes()

ISourceType[] org.eclipse.jdt.internal.compiler.env.NameEnvironmentAnswer.getSourceTypes ( )

Answer the unresolved source forms for the type or null if the receiver represents a compilation unit or binary type.

Multiple source forms can be answered in case the originating compilation unit did contain several type at once. Then the first type is guaranteed to be the requested type.
