org.eclipse.jdt.internal.core.dom.rewrite.imports.ImportRewriteConfiguration.ImportContainerSorting.createContainerComparator
abstract Comparator< ImportName > createContainerComparator(JavaProject javaProject)

Sorts imports by each import's package, in lexicographic order. For example (assuming all of the imports belong to the same import group): 

import java.net.Socket;
import java.util.Map;
import java.util.Map.Entry;
import java.util.Set;

import java.net.Socket;
