Detailed Description

Standard implementation of IDOMFactory, and the only means of creating JDOMs and document fragments.

This class may be instantiated.

Deprecated:
The JDOM was made obsolete by the addition in 2.0 of the more powerful, fine-grained DOM/AST API found in the org.eclipse.jdt.core.dom package. @noextend This class is not intended to be subclassed by clients.
