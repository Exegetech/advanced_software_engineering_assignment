Detailed Description

A source range defines an element's source coordinates relative to its source buffer.

Clients may use the method org.eclipse.jdt.core.SourceRange#isAvailable(ISourceRange) in order to find out if a source range is available. This method returns false when the source range offset is equals to -1.

Clients may use the default implementation provided by SourceRange.

See also
SourceRange @noimplement This interface is not intended to be implemented by clients.
