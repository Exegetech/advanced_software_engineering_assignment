Detailed Description

A source element parser extracts structural and reference information from a piece of source.

also see @ISourceElementRequestor

The structural investigation includes:

  • the package statement
  • import statements
  • top-level types: package member, member types (member types of member types...)
  • fields
  • methods

If reference information is requested, then all source constructs are investigated and type, field & method references are provided as well.

Any (parsing) problem encountered is also provided.
