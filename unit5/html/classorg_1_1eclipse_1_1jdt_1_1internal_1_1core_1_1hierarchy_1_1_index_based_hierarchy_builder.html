Find the set of candidate subtypes of a given type.

The requestor is notified of super type references (with actual path of its occurrence) for all types which are potentially involved inside a particular hierarchy. The match locator is not used here to narrow down the results, the type hierarchy resolver is rather used to compute the whole hierarchy at once.
