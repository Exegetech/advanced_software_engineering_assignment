Detailed Description

A java element delta builder creates a java element delta on a java element between the version of the java element at the time the comparator was created and the current version of the java element.

It performs this operation by locally caching the contents of the java element when it is created. When the method createDeltas() is called, it creates a delta over the cached contents and the new contents.
