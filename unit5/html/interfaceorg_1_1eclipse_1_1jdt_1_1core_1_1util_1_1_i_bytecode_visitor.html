Detailed Description

Description of a Java opcodes visitor. This should be used to walk the opcodes of a ICodeAttribute.

Clients must subclass ByteCodeVisitorAdapter to define an implementation of this interface.

Since
2.0 @noimplement This interface is not intended to be implemented by clients.
