CategorizedProblem[] org.eclipse.jdt.internal.compiler.CompilationResult.getProblems ( )

Answer the problems (errors and warnings) encountered during compilation.

This is not a compiler internal API - it has side-effects ! It is intended to be used only once all problems have been detected, and makes sure the problems slot as the exact size of the number of problems.
