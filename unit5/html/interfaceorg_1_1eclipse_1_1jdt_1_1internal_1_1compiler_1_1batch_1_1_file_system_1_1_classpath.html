Detailed Description

A Classpath, even though an IModuleLocation, can represent a plain classpath location too. The FileSystem tells the Classpath whether to behave as a module or regular class path via Classpath#acceptModule(IModule).

Sub types of classpath are responsible for appropriate behavior based on this.
