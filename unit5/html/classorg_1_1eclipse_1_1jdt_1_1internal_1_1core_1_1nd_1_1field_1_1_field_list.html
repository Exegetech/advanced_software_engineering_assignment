Detailed Description

Stores a singly-linked list of blocks, each of which contains a variable number of embedded elements. Each block contains a header containing the size of the block and pointer to the next block, followed by the embedded elements themselves.

Ensures that the receiver will have space for the given number of elements without additional allocation. Callers should invoke this prior to a sequence of FieldList#append(Nd, long) calls if they know in advance how many elements will be appended.
