Detailed Description

Wrapper used to link IRestrictedAccessMethodRequestor with MethodNameRequestor. This wrapper specifically allows usage of internal method BasicSearchEngine#searchAllMethodNames( char[] packageName, int pkgMatchRule, char[] declaringQualification, int declQualificationMatchRule, char[] delcaringSimpleName, int declSimpleNameMatchRule, char[] methodName, int methodMatchRule, IJavaSearchScope scope, IRestrictedAccessMethodRequestor methodRequestor, int waitingPolicy, IProgressMonitor progressMonitor) from API method org.eclipse.jdt.core.search.SearchEngine#searchAllMethodNames( char[] packageName, int pkgMatchRule, char[] declaringQualification, int declQualificationMatchRule, char[] delcaringSimpleName, int declSimpleNameMatchRule, char[] methodName, int methodMatchRule, IJavaSearchScope scope, MethodNameRequestor methodRequestor, int waitingPolicy, IProgressMonitor progressMonitor).
